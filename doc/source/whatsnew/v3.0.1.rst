.. _whatsnew_301.bugs:

Bug fixes
^^^^^^^^^
- Fixed a bug in the :class:`DataFrame` constructor when passed a :class:`Series` or
  :class:`Index` correctly handling Copy-on-Write (:issue:`63899`)
- Fixed a bug in :func:`testing.assert_frame_equal` and :func:`testing.assert_series_equal` where comparing nested ``list``, ``tuple``, or pandas ``ExtensionArray`` with ``numpy.ndarray`` containing the same values incorrectly raised ``AssertionError`` (:issue:`63904`)
